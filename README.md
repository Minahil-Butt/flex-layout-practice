# 📐 Flexbox Layout Practice

This project focuses on building responsive layouts using **CSS Flexbox**. It includes multiple mini-projects like pricing cards and team profile sections to strengthen layout skills.

## 📄 Project Overview

A collection of Flexbox-based UI components:
- 💳 Pricing Card Layout
- 👥 Team Profile Cards
- 📱 Responsive design using Flexbox

Each section demonstrates alignment, spacing, and flexible layouts using modern CSS techniques.

## 🛠️ Built With

- HTML5  
- CSS3 (Flexbox)  
- No JavaScript used  

## 🧠 Key Learnings

- Understanding Flexbox properties (`flex`, `justify-content`, `align-items`)  
- Building responsive layouts  
- Structuring reusable UI components  
- Improving alignment and spacing techniques
